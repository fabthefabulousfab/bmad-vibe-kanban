# Story 1-1/0: Brainstorm Ideas

**Wave:** 1 | **Epic:** 1 | **Story:** 0

## User Story

**As a** solo developer starting a new project,
**I want** to brainstorm and capture initial ideas,
**So that** I have a clear starting point for product development.

## Acceptance Criteria

1. [ ] Initial ideas documented
2. [ ] Problem statement defined
3. [ ] Target user identified
4. [ ] Key differentiators listed

**Command:**
```bash

```

**Input:** Initial product idea
**Output:** `_bmad-output/brainstorm/ideas.md`
**Validation:** Document contains problem, user, and differentiators
