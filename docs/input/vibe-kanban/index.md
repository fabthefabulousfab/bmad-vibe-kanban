### Quick Reference

#### Frontend (React SPA)
- **Tech Stack:** React 18, Vite, TailwindCSS, Zustand, TanStack Query
- **Entry Point:** `frontend/src/main.tsx`
- **Root:** `/frontend`

#### Backend (Rust API)
- **Tech Stack:** Axum, Tokio, SQLx, SQLite
- **Entry Point:** `crates/server/src/main.rs`
- **Root:** `/crates`

#### Shared Types
- **Tech Stack:** ts-rs generated TypeScript
- **Entry Point:** `shared/types.ts`
- **Root:** `/shared`

#### NPX CLI
- **Tech Stack:** Node.js wrapper
- **Entry Point:** `npx-cli/bin/cli.js`
- **Root:** `/npx-cli`

## Getting Started

### For Development

```bash
# 1. Install dependencies
pnpm install

# 2. Start development servers
pnpm run dev

# 3. Open browser (ports shown in terminal)
```

See [Development Guide](./development-guide.md) for detailed setup.

### For AI-Assisted Development

When using AI agents to work on this codebase:

1. **Provide context:** Point to this index and relevant architecture docs
2. **For frontend changes:** Reference `frontend/CLAUDE.md` for styling rules
3. **For backend changes:** Reference `architecture.md` for patterns
4. **For type changes:** Run `pnpm run generate-types` after Rust changes
