# Vibe Kanban - Project Overview

> AI coding agent orchestration and task management platform

## Executive Summary

Vibe Kanban is a full-stack application that enables developers to orchestrate multiple AI coding agents (Claude Code, Gemini CLI, OpenAI Codex, etc.) from a centralized Kanban-style interface. It streamlines the process of planning, executing, and reviewing AI-generated code changes across multiple tasks in parallel.

## Core Capabilities

1. **Multi-Agent Orchestration** - Switch between and run multiple AI coding agents
2. **Parallel Task Execution** - Execute multiple tasks simultaneously with isolated git worktrees
3. **Code Review Interface** - Review diffs, add comments, create PRs
4. **Task Management** - Kanban board for tracking AI agent work
5. **MCP Server Configuration** - Centralized Model Context Protocol configuration
6. **Remote SSH Support** - Open projects remotely via SSH tunnels

## Architecture Pattern

**Layered Full-Stack Architecture** with:
- **Presentation Layer**: React SPA with component-based UI
- **API Layer**: Axum REST + WebSocket endpoints
- **Service Layer**: Rust business logic in `services` crate
- **Data Layer**: SQLite with SQLx for local, PostgreSQL for remote
- **Integration Layer**: Agent executor abstractions for multiple AI providers

## Key Integration Points

1. **Frontend ↔ Backend**: REST API + WebSocket via Vite proxy
2. **Type Sharing**: ts-rs generates `shared/types.ts` from Rust structs
3. **Agent Communication**: Executor layer abstracts agent-specific protocols
4. **Git Operations**: git2 manages worktrees for isolated task execution
